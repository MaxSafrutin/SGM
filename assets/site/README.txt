assets/site — бренд и “одна точка правды”

Здесь лежат файлы, которые используются по всему сайту:

- logo.svg  — основной логотип (используется на главной странице).
  Чтобы заменить логотип, просто замени этот файл своим (с тем же именем).

- icon.svg  — мини‑логотип для иконки сайта (favicon / apple-touch-icon).
  Чтобы заменить иконку, замени этот файл своим (с тем же именем).

Рекомендация по ассетам:
- Картинки для слайдов: assets/media/
- PDF/файлы для скачивания: assets/downloads/
